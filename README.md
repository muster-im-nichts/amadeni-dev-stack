# Amadeni Dev Stack

Architecture, conventions, and tooling for agent-driven software development at Amadeni.

This repository documents the system that produces our software — not the software itself. It evolves week by week as we learn what works.

## What's Inside

- **`stack.md`** — The current state of the Amadeni tech stack (versioned snapshot)
- **`conventions/`** — AGENTS.md snippets ready to copy into project repos
- **`changelog.md`** — Week-by-week evolution log
- **`ideas/`** — Improvement proposals not yet implemented

## Philosophy

Agents write the code. Humans set the direction. This repo bridges the two by defining the rules, patterns, and tools that make agent-driven development reliable and predictable.

The goal: a client project can be scaffolded, developed, tested, and reviewed with minimal human intervention — and the result is code you'd be proud to ship.
